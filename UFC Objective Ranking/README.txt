In this project I collected data from the website website UFCSTATS.com and used them to rank fighters in each weight class in an ELO system.
Some intersting facts and data like finish rate each year, most common way to get KO'd or submitted are also included.
These stats were then visualized on a web application using the streamlit library in python.
DISCLAIMER: The front end part of this project was handled using AI, this was done to accelarate development.
