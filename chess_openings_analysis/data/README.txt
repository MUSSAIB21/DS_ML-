Please download the chess dataset manually:
            1. Go to: https://www.kaggle.com/datasets/shkarupylomaxim/chess-games-dataset-lichess-2017-may
            2. Click 'Download' button
            3. Extract the zip file
            4. Place 'games_metadata_profile_2024_01.csv' in your 'data/' folder
            Your data folder should look like:
              data/
                └── games_metadata_profile_2024_01.csv
